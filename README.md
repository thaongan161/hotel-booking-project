# Hotel Booking System

Full-stack hotel booking application:

- **Backend**: Spring Boot + Maven + MySQL + JWT + Spring Security  
- **Frontend**: React + Vite + TypeScript + TailwindCSS + Framer Motion  

---

## Requirements

- Java 24  
- Maven 4+  
- Node.js 20+ & npm 9+  
- MySQL 8+  

---

## Build
cd backend
mvn clean install

## Build
cd frontend
npm install

## Run fullstack
./run.bat
